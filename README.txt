Hell's A Kitchen README
~~~~~~~~~~~~~~~~~~~~~~~~~
A Game Developed by: Zach Spindel, Lindsey Pettibon, Ares Wong, Joshua Garcia, and Diego Majian

For GMAP-345-001, Fall Quarter 2024

~~~~~~~~~~~~~~~~~~~~~~~~~

Controls:

[Arrow Keys] or [WASD] to move
[L-Shift] or [R-Shift] to Sprint
[E] or [I] to Interact
[F] or [J] to Cut
[O] to Cast Fireball (needs kitchen object as source)
[B] to Build Turrets
